# v0.4.0 更新公告

### 修复与优化

- 修复了一系列问题 (大概吧QAQ)
- 提升了跑图时点击的速度
- **[重要]** 现在的“清空数据”变成了一个单独任务。中途停止再继续，吃药数量不会重新计数啦。

### 新功能：自动跳转

新增“前往指定页面”功能，配合游戏自动启动和任务自动启动的设置,甚至能做到双击软件就能开始挂机，不用手动跳到地图页了！
*当前支持：日常图、星海一*

---

### 💡 隐藏玩法 Tip

**左侧的任务是可以点击 `+` 号重复添加的哦！**

**一条龙挂机配置示例：**

1. `前往日常地图`
2. `跑图` (检测到日常图跑完回到选图界面后，此任务会自动结束)
3. `前往星海`
4. `跑图`

你甚至可以在中间穿插一个 `竞技场`，彻底解放双手。
